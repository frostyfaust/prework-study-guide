HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Style defines the CSS styles associated with the website.
  • h1 - h6 provide heading levels for given text.
  • br creates a line break.
  • ul (unordered list), ol (ordered list), and li (listen item) allow us to make different types of lists and items within those lists.

CSS

    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • Inline CSS apply to a specific html tag and is useful for quick styling and changes.
    • Internal CSS style sheet that contains CSS rules for the page, usually in the head section. Only applies to the specific page.
    • External CSS is mostly preferred. Uses CSS rules in a seperate file that can get implimented into other webpages.
    • different elements can utilize different classes to help style them more easily.

Git

  • Creating New Branches
    • git checkout -b [name of branch] - syntax for creating new branch off a main branch.
    • git add -A - syntax used to add and update the branch you're working on.
    • git commit -m "description of what changed" - syntax used to commit and add comments of what changed.
    • git pull origin main - syntax used to let us see if we have the latest commits and changes.
    • git push origin [name of branch you're using] - syntax used to push changes to github using the name of the branch you're using. NEVER USE MAIN BRANCH OF THIS!

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A String is a sequence of text tied with quotations.
  • A Boolean is a true or false value.
  • An Array is a structure that allows you to share multiple values in a single reference.
  • Linking JS style sheets us this syntax: [script src="Js file path"] [/script]. normally link the JS style sheet at the end of the body element.
  • If statements are code that will only execute if the statement is true or meets the conditions needed.
  • For loops use a predictable pattern of indices to perform a task on the items in an array.
